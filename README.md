RecursionCSのBackendProject2_Remote-Procedure-Callのリポジトリーになります。

## 概要
TCP通信を用いてクライアント、サーバ間でのデータ送受信(JSON形式)を行います。

クライアントは、Javascript(Node.js)、サーバはpythonを使用しRPCのシステムを実装しています。
